# mqtt2js
Linux virtual joystick controlled by MQTT

# Building
On Ubuntu:

    sudo apt install build-essential pkg-config libmosquitto-dev libevdev-dev
    make

# Command line usage
The complete command-line usage help is available by typing

    mqtt2js -h
